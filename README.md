Overview

Simple E-Commerce Store – A Full Stack Web Application for Online Shopping Experience

Introduction

This project is a full-stack e-commerce web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. HTML is used to structure the web pages, CSS is used for styling and responsive design, and JavaScript is used for dynamic user interactions. The backend is developed using Node.js and Express.js to handle APIs, authentication, product management, and order processing. MongoDB is used as the database to store users, products, and orders.

The application is designed to provide a smooth and interactive online shopping experience with secure authentication and efficient cart and order management features.

Core Features of E-Commerce Website
User Authentication

The system supports user registration and login functionality with secure authentication using JWT. Passwords are encrypted using bcrypt to ensure data security. Protected routes are implemented to restrict unauthorized access.

Product Management

The application displays a list of products with images, descriptions, and prices. Users can search and filter products based on categories. Each product has a detailed view page with complete information.

Shopping Cart System

Users can add products to the cart, update quantities, and remove items. The cart automatically calculates the total price and maintains updated cart data for a smooth shopping experience.

Order Processing

The checkout system allows users to enter shipping details and place orders. Order summaries are displayed before confirmation, and all order data is stored in the database. Users can also view their order history.
